# Simulation de la photoluminescence résolue en temps (TRPL) dans un semi-conducteur

##Description 

Ce projet modélise la dynamique de recombinaison des porteurs générés par une impulsion laser dans un semi-conducteur. L'objectif est d'étudier l'influence des différents mécanismes de recombinaison sur le signal de photoluminescence résolue en temps (TRPL).
